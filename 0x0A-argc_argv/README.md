# 0x0A. C - Command Line Arguments (argc & argv)

This project is part of the WinMingle Community Software Engineering Program.

## Description
This project focuses on using command-line arguments in C programs through
the parameters `argc` and `argv`.

## Learning Objectives
- Understand what `argc` and `argv` mean
- Learn how command-line arguments are passed to `main`
- Practice input validation in C
- Use standard library functions like `printf` and `atoi`

## Requirements
- Allowed editors: vi, vim, emacs
- Compiled on Ubuntu 20.04 LTS
- Compiler flags: `-Wall -Werror -Wextra -pedantic -std=gnu89`
- No global variables
- No more than 5 functions per file
- All files end with a new line

## Files
- `0-whatsmyname.c` – Prints the program name
- `1-args.c` – Prints the number of arguments
- `2-args.c` – Prints all arguments
- `3-mul.c` – Multiplies two numbers
- `4-add.c` – Adds positive numbers
- `100-change.c` – Calculates minimum number of coins
