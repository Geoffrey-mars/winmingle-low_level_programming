# 0x08. C - Recursion

## Learning Objectives

At the end of this project, you should be able to explain:

- What recursion is
- How to implement recursion in C
- When to use recursion
- When not to use recursion

---

## Requirements

- Allowed editors: `vi`, `vim`, `emacs`
- All files compiled on Ubuntu 20.04 LTS with:
gcc -Wall -Werror -Wextra -pedantic -std=gnu89

- All files should end with a new line
- No global variables
- Maximum 5 functions per file
- Do not use loops or static variables
- Do not use standard library functions (e.g., `printf`, `puts`, `strlen`)
- You may use `_putchar`
- Function prototypes must be in `main.h`
- Push `main.h` and source files to your repository

---

## Tasks

|------|------|-----------|-------------|
