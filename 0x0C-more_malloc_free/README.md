# 0x0C. C - More malloc, free

## Description
This project is part of the WinMingle Software Engineering Program.
It focuses on advanced dynamic memory allocation in C using `malloc`,
`free`, and `exit`.

## Learning Objectives
By completing this project, I learned:
- How to use `malloc` safely
- How to terminate a program using `exit`
- How to manually implement `calloc` and `realloc`
- How to manage memory without leaks

## Requirements
- Ubuntu 20.04 LTS
- gcc with flags: `-Wall -Werror -Wextra -pedantic -std=gnu89`
- No global variables
- No more than 5 functions per file
- Only `malloc`, `free`, and `exit` allowed in solutions
